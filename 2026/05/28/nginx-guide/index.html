一、Nginx 简介

Nginx（engine x）是一个高性能的 HTTP 和反向代理服务器，由 Igor Sysoev 于 2004 年发布。它以高并发、低内存消耗、高稳定性著称，是全球使用最广泛的 Web 服务器之一。

核心特性

  • 事件驱动架构：基于 epoll/kqueue 的异步非阻塞模型
  • 反向代理与负载均衡：天然支持多种负载均衡策略
  • 静态资源服务：极高的静态文件处理能力
  • 模块化设计：功能通过模块扩展，灵活可定制

与 Apache 的对比

特性 Nginx Apache
并发模型 事件驱动，异步非阻塞 进程/线程模型
内存占用 低 相对较高
静态文件 极快 一般
动态内容 通过 FastCGI/反向代理 内置 mod_php 等
配置风格 集中式配置 分布式 .htaccess

总结
